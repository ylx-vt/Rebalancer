# Rebalancer 需求行动计划

## 1. 规划目标
在基础再平衡观测能力完成后，逐步补充面向“规则验证”和“配置比较”的高级功能。

后续需求继续遵循当前产品边界：
1. 产品形态仍为 Chrome 插件。
2. 核心体验仍在 Popup 内完成。
3. 不做交易执行，不做自动调仓，不做投资建议。
4. 所有回测结果必须口径清晰、可解释。

---

## 2. 行动清单

| 阶段 | 需求 | 状态 | 目标 |
| --- | --- | --- | --- |
| 阶段一 | 再平衡规则回测 | 待规划 | 对比不同阈值、不同再平衡频率下的历史表现 |
| 阶段二 | 持仓比例回测对比 | 待规划 | 对比不同持仓比例配置带来的历史表现差异 |

---

## 3. 阶段一：再平衡规则回测

### 3.1 需求目标
在同一套持仓配置下，比较不同再平衡规则的历史表现，帮助用户判断自己的再平衡触发方式是否合理。

### 3.2 对比维度
1. 不再平衡。
2. 固定频率再平衡：
   - 月度
   - 季度
   - 半年
   - 年度
3. 阈值触发再平衡：
   - 偏离达到指定百分点后再平衡
   - 相对目标比例偏离达到指定比例后再平衡
4. 频率与阈值组合：
   - 按固定频率检查
   - 仅在达到阈值时执行再平衡

### 3.3 核心输出
1. 期末金额。
2. 累计收益率。
3. 年化收益率。
4. 最大回撤。
5. 波动率。
6. 再平衡次数。
7. 调仓总金额。
8. 收益曲线。
9. 回撤曲线。
10. 调仓记录。

## 4. 阶段二：持仓比例回测对比

### 4.1 需求目标
在相同回测区间和相同再平衡规则下，对比不同持仓比例配置的历史表现，帮助用户理解资产配置比例变化带来的收益与风险差异。

### 4.2 对比维度
1. 不同基金组合。
2. 相同基金、不同目标比例。
3. 是否保留现金仓位。
4. 不同配置在同一规则下的收益、回撤、波动、调仓差异。

### 4.3 核心输出
1. 配置对比表。
2. 收益曲线对比。
3. 回撤曲线对比。
4. 风险收益指标对比。
5. 调仓行为对比。

### 4.4 关联需求文档
[持仓比例回测对比需求文档](../需求文档/持仓比例回测对比需求文档.md)

## 5. 优先级建议
1. 优先完成阶段一，再做阶段二。
2. 阶段一先解决历史净值序列、日期对齐、再平衡模拟和结果展示。
3. 阶段二复用阶段一的回测能力，只新增多配置对比入口和结果展示。
4. MVP 阶段不引入自动参数优化，避免产品变成复杂策略平台。
